💾 Backup & Restore

Your data is stored in the cloud (Firebase). Backups are optional but recommended for extra safety.

How it works:

Cloud sync: All data is automatically saved to Firebase Firestore. You can access it from any device.

Export downloads a .json backup file. Save it to Google Drive or OneDrive for extra safety.

Import restores data from a previously exported file. Data is merged with existing data (no duplicates).

Offline mode: The app works offline using local storage and syncs to cloud when back online.
